# ChurnApp: Predicción y visualización de fuga de clientes
## Descripción breve
Este proyecto consiste en una aplicación web de extremo a extremo que permite predecir el riesgo de churn en el sector de las telecomunicaciones. Integra un backend en Python, encargado del modelado y la generación de predicciones, y un frontend que ofrece un panel de control sencillo para visualizar métricas globales, distribución de riesgos y detalles explicativos por cliente o segmento.
## Objetivos principales
- Desarrollar un backend en Python que implemente modelos de machine learning previamente validados para la predicción de churn.
- Diseñar un frontend interactivo que muestre de manera clara los resultados, con métricas globales y explicaciones interpretables a nivel individual.
- Transformar los datos en información accionable, permitiendo priorizar clientes en riesgo y comprender las causas principales de su posible abandono.
- Proporcionar un ejemplo práctico de integración end-to-end en ciencia de datos, desde la ingesta y procesamiento de información hasta la visualización para usuarios no técnicos.
## Plan inicial de trabajo
- **Fase 1: Preparación de datos y entrenamiento de modelos**  
Revisión del dataset y generación de modelos.  

- **Fase 2: Desarrollo del backend**  
Implementación de API en FastAPI para predicciones y explicaciones.  

- **Fase 3: Desarrollo del frontend**  
Creación de un panel de control con métricas y visualización por cliente.  

- **Fase 4: Integración y despliegue**  
Conexión frontend-backend, pruebas finales y documentación.
